What is it?

Enables to open content in an interactive overlay. May load content that is already on the page, via AJAX or programmatically. Multiple modal boxes can be opened at the same time.

Features

  • Multiple modal boxes can be opened at the same time.
  • Multiple styles can be used on the same page.
